1️⃣ Difference between var, let, and const:
Ans:
var → Function scoped, can be re-declared and updated.
let → Block scoped, can be updated but cannot be re-declared in the same scope.
const → Block scoped, cannot be updated or re-declared (constant value).

2️⃣ What is the spread operator (...)?
Ans:
The spread operator expands elements of an array or object into individual elements.

3️⃣ Difference between map(), filter(), and forEach():
Ans:
map() → Creates a new array by transforming each element.
filter() → Creates a new array with elements that match a condition.
forEach() → Loops through array but does not return a new array.

4️⃣ What is an arrow function?
Ans:
A shorter syntax for writing functions introduced in ES6.

5️⃣ What are template literals?
Ans:
Template literals allow embedding variables and expressions inside strings using backticks `.
